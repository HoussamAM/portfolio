# Houssame-Eddine Ait Moulay — Portfolio

A cinematic, multi-stage personal portfolio website built with pure HTML, CSS, and vanilla JavaScript. No frameworks, no build tools — just hand-crafted front-end work.

🔗 **Live site:** [houssamam.github.io/portfolio]

---

## ✨ Experience Flow

The site is designed as a sequential cinematic experience:

1. **Terminal** — A fake bash terminal types out commands, introducing the portfolio.
2. **Game Library** — A PS5-style game library UI with hero art and a LAUNCH button.
3. **Rockstar Intro** — A GTA Rockstar-inspired intro with a logo and loading bar.
4. **Portfolio** — The actual portfolio: projects, skills, languages, and contact.

## 🚀 Features

- **4-stage cinematic intro** — Terminal → Library → Rockstar Intro → Portfolio
- **Weapon Wheel overlay** — GTA-style SVG weapon wheel for Projects & Skills
- **Multilingual greeting** — Cycles through Arabic, English, French, German, Tamazight, and Cyberpunk
- **GTA Wanted stars** — Used as proficiency indicators for the Languages section
- **Web Audio API** — Subtle UI beep sounds on interactions
- **IntersectionObserver animations** — Smooth scroll-entrance effects on all sections
- **Dark/Light theme toggle** — Accessible theme switcher in the nav
- **Responsive** — Works across desktop and mobile viewports
